חיבור Twilio ללקוח

יש להזין מספר מלא עם קידומת מדינה. לאחר השמירה המערכת תעדכן אוטומטית את היחידה של הלקוח.
הערך נמדד במילישניות: 1000 = שנייה אחת, 500 = חצי שנייה, 0 = ללא המתנה. מומלץ להתחיל עם 500-1000.
-
את הכתובת הזו מדביקים ב-Twilio בשדה Webhook URL for incoming messages, עם HTTP POST.
כל לקוחה מקבלת שירות ואחסון נפרדים. העמוד הזה יציג סטטוס אמת וקישור כניסה, ללא גישה לנתונים של לקוחות אחרות.
